# Aikyam Ananda Pre-School

Welcome to the official website repository for **Aikyam Ananda Pre-School**. Built with modern web development practices, this website showcases the school's vision, holistic development approach, faculty details, and events, while providing an easy-to-use admission enquiry form for prospective parents.

## 🌟 Features

- **Playful & Vibrant Design:** Aesthetics tailored for a pre-school audience with candy floss styling, blob cards, and engaging animations.
- **Responsive Layout:** fully responsive across mobile, tablet, and desktop devices.
- **Holistic Development Info:** Cards detailing Emotional Growth, Cognitive Skills, Physical Activity, and Social Skills.
- **Enquiry Form Modal:** A detailed admission application modal ensuring parents can easily reach out.

## 🛠️ Technology Stack

- **HTML5:** Semantic structuring of the content.
- **CSS3:** Custom styling utilizing CSS variables, animations, and clean grid/flexbox layouts. 
- **Vanilla JavaScript (`script.js`):** Scripting for interactive elements such as modals and form handling.

## 🚀 Getting Started

1. Clone the repository.
2. Open `index.html` in any modern web browser or use an extension like **Live Server** in VS Code to run the project locally.
